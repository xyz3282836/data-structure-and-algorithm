# 内排序

### 直接插入排序

```
public InsertSort() {
    int a[] = {49, 38, 65, 97, 76, 13, 27, 49, 78, 34, 12, 64, 5, 4};
    int temp = 0;
    for (int i = 1; i < a.length; i++) {
        int j = i - 1;
        temp = a[i];
        for (; j >= 0 && temp < a[j]; j--) {
            a[j + 1] = a[j];//将大于temp的值整体后移一个单位
        }
        a[j + 1] = temp;
    }
}
```
